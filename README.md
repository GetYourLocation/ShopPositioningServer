# ShopPositioningServer

Server that positionings shops.

## Installation

- Requirements

    - MATLAB R2017a

- Run server

    - Terminal

        ```
        $ ./run.sh
        ```

    - MATLAB

        ```matlab
        >> server
        ```

- Run client (for testing)

    ```matlab
    >> client
    ```

- Run training program

    ```matlab
    >> train
    ```

- Run testing program

    ```matlab
    >> test
    ```

- Predict one image

    ```matlab
    >> predictOne(path_to_image)
    ```
