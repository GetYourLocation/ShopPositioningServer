# ShopPositioningServer

Server that positionings shops.

## Installation

- Requirements

    - MATLAB R2017a

- Run server

    ```matlab
    >> server
    ```

- Run client (for testing)

    ```matlab
    >> client
    ```

- Run training program

    ```matlab
    >> train
    ```
